# Used Car Sales Dashboard

This project explores used car sales data from the US and turns it into an interactive analytical dashboard.  
Built with **Python**, **Plotly**, and **Streamlit**, it helps users understand how vehicle characteristics affect price and time on market.

 Full project repository:  
https://github.com/Pushkarski/05_used_car_sales_dashboard

### Project Highlights
- Exploratory data analysis (EDA) of US vehicle listings  
- Study of how condition, mileage, model year, and fuel type affect price  
- Interactive Streamlit dashboard with multiple visualizations  
- Insights for understanding pricing patterns and market behavior
